MAP HELP


Click any location pin on the map to display more information about that event.

TOP RIGHT: Click on the my location button located to return to your original location.

RIGHT: Use the filter/key button to toggle what type of events are visibile on the map, or to view the map key.

BOTTOM RIGHT: Use the load more button to display more events in the map.

TOP LEFT: Use the search button to search for any city or address in the world.

LEFT:Use the events button to see a drop down list of events. Upon clicking on an event, the map will zoom to that location. The pink badge indicates the number of events displayed in the map
